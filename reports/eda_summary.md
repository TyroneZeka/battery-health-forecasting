# Battery RUL Project - EDA Summary

## Dataset Overview
- Total Records: 4870
- NASA Batteries: 34
- Oxford Batteries: 8
- Max Cycles (NASA): 614
- Max Cycles (Oxford): 2076

## Key Insights
- Oxford batteries show more variability in degradation.
- NASA batteries have a more consistent, linear degradation pattern.
- RUL prediction is a viable goal for both datasets.
